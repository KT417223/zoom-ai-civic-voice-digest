# Sources

このプロジェクトで参照する公開音声ページの候補です。

## 藤沢市「声の広報ふじさわ」

- URL: https://www.city.fujisawa.kanagawa.jp/kouhou/shise/koho/kohofujisawa/koe/podcast.html
- メモ: 各号が複数のMP3ファイルに分かれている。2025年8月10日号以降、一部でVOICEVOX NemoのAI音声を使用している旨の記載がある。
- 記事での使い方: 元ページへのリンクと処理結果の短い例を掲載する。音声ファイル自体は再配布しない。

## 小平市「市報音声版『声のたより』」

- URL: https://www.city.kodaira.tokyo.jp/shihou-voice/
- メモ: 音声ファイル形式がMP3であること、ストリーミング版とダウンロード版があることが明記されている。
- 記事での使い方: MP3形式の公開音声例として参照する。

## 所沢市「広報ところざわ 音声版」

- URL: https://www.city.tokorozawa.saitama.jp/tokoronews/koho/onsei/index.html
- メモ: 音訳ボランティアの録音を活用し、毎号5から6ファイルに分割して掲載している。
- 記事での使い方: 長めの音声を要約する比較候補として参照する。

## 立川市「広報たちかわ（PDF版）・声の広報（音声版）」

- URL: https://www.city.tachikawa.lg.jp/shisei/koho/1005488/1005547/index.html
- メモ: 2020年7月10日号から「声の広報」を公開している。
- 記事での使い方: バックナンバーが豊富な公開例として参照する。
